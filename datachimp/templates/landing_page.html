Data versioning made easy


No need to maintain multiple files or folders for your data science or machine learning use case

How to get Started?

Step 1: Create the project
Step 2: Copy the key
Step 3: Install datachimp using “pip install datachimp” and use the following command to sync your data

$> datachimp sync -k <project_key> -h datachimp.modelchimp.com <file_or_folder>

# Example

$> datachimp sync -k 418b3212-a9b2-4643-a62c-0e0326577be5 -h datachimp.modelchimp.com train.csv

Step 4: View the data in the DataChimp Portal
Step 5: To Pull the data back to your system, use the following command

$> datachimp pull-data -k <project_key> -h datachimp.modelchimp.com <data_version>

# Example

$> datachimp pull-data -k 418b3212-a9b2-4643-a62c-0e0326577be5 -h datachimp.modelchimp.com c0140d49a2e546e7e5bf7d670fa3b8e3868e4c60fee2d20ccc21052a8bc31517
